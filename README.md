# Vapor Prompt Input Layer

Vapor UI를 사용해 AI 챗 입력 영역을 재사용 가능한 제품 컴포넌트 레이어로 설계한 케이스 스터디입니다.

PromptBox, Dropzone, AttachmentList, DataSourceSelector를 제품 기준 컴포넌트로
정의하고, 앱 레이어가 Vapor primitive를 직접 사용하지 않도록 경계를 강제합니다.

## Tech Stack

React · TypeScript · Vite · Vapor UI · Vitest · Playwright

## Development

```bash
npm install
npm run dev
```
